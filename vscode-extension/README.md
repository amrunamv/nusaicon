# NusaIcons VS Code Extension

Provides intelligent autocomplete and hover previews for the official NusaIcons SVG library.

## Features

- Autocompletes `icon-*` class names inside your HTML, JSX, TSX, Vue, and PHP files.
- Automatically documentation link for each icon.
