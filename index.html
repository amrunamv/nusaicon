Dokumentasi NusaIcons

1. Instalasi (CDN)

Untuk mulai menggunakan ikon, masukkan tag <link> berikut ke dalam blok <head> dokumen HTML Anda:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/amrunamv/nusaicon@main/nusaicons.css">

2. Cara Penggunaan

Gunakan tag <i> atau <span> dengan class awalan icon-{nama-ikon}. Ikon akan otomatis seukuran dengan teks di sekitarnya karena menggunakan satuan 1em.

Contoh Dasar:

<button> <i class="icon-download"></i> Unduh File </button>

3. Mewarnai & Mengubah Ukuran

NusaIcons dibuat menggunakan teknik CSS Masking, sehingga ikon akan selalu mewarisi warna dari atribut color milik induknya (currentColor).

Via Tailwind CSS

<i class="icon-heart text-rose-500 text-3xl"></i>

Via CSS Biasa

.icon-heart {
  color: #f43f5e;
  font-size: 30px;
}
